查看提交历史

通过git log命令查看历史，可添加不同参数来显示需要的历史内容。

  • -p 查看每次提交的内容差异，-2 显示最近的两次更新

  • --stat 摘要显示修改的文件以及代码删减多少行

  • --pretty=... 可以指定使用完全不同于默认格式的方式查看提交历史。

oneline — 将每个提交放在一行显示

full、fuller、format —定时要显示的记录格式

--graph 用oneline 或 format 时结合 --graph选项，一些简单图形展示了每个提交所在的分支及其分化衍合情况。

查看命令历史

git reflog 查看命令历史，以便确定要回到未来的哪个版本。
