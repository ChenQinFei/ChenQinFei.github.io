  • offsetWidth

    е…ғзҙ еңЁеһӮзӣҙж–№еҗ‘дёҠеҚ з”Ёзҡ„з©әй—ҙеӨ§е°ҸпјҢеҢ…жӢ¬е…ғзҙ й«ҳеәҰгҖҒпјҲеҸҜи§Ғзҡ„пјүж°ҙе№іж»ҡеҠЁжқЎй«ҳеәҰгҖҒдёҠдёӢиҫ№жЎҶй«ҳеәҰ

  • offsetLeft

    е…ғзҙ зҡ„е·ҰеӨ–иҫ№жЎҶиҮіеҢ…еҗ«е…ғзҙ пјҲoffsetParentпјүзҡ„е·ҰеҶ…иҫ№жЎҶд№Ӣй—ҙзҡ„еғҸзҙ и·қзҰ»

  • offsetTop
